## TC-003: 自動保存

### TC-003-01: debounce動作
- useScene変更後、5秒後に自動保存される
- 5秒以内の連続変更はまとめて1回だけ保存される

### TC-003-02: 保存中の二重保存防止
- 保存処理中に新たな保存は行わない
- 保存完了後に変更があれば再度保存する

### TC-003-03: エラーリトライ
- 保存エラー時にリトライする
- リトライ後も失敗した場合はエラー状態になる
