## 実装詳細

### 1. ポリゴン頂点クリック・プレビュー 🔵

**信頼性**: 🔵 *dataflow.md ゾーン描画フロー*

3Dビュー上のクリックで [x,z] 座標を取得。ローカルstate に頂点配列を保持。半透明ポリゴンでリアルタイムプレビュー（シーン保存しない）。

### 2. Enter確定・ノード作成 🔵

**信頼性**: 🔵 *REQ-102 + ツール規約*

Enter → HvacZoneNode.parse({ boundary, usage, floorArea }) → useScene.createNode(zone, levelId)。面積は boundary から自動算出。

### 3. ゾーン属性入力ダイアログ 🔵

**信頼性**: 🔵 *REQ-102 + dataflow.md*

確定後にゾーン名、用途(ZoneUsage)、設計条件(DesignConditions)の入力ダイアログ。

**実装ファイル**: `apps/kuhl-editor/components/tools/zone-draw-tool.tsx`
