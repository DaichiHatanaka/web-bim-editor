## 実装詳細

### 1. LOD100 ボックスレンダリング 🔵

**信頼性**: 🔵 *architecture.md レンダラーパターン*

node.dimensions([w,h,d])→BoxGeometry。equipmentColor マップ（ahu=青、pac=緑、fcu=橙等）。position, rotation 適用。

### 2. TagLabel コンポーネント 🔵

**信頼性**: 🔵 *REQ-206*

Html（drei）でタグ名を3Dビュー上に表示。機器上方にオフセット。

**実装ファイル**: `packages/kuhl-viewer/src/components/renderers/parts/tag-label.tsx`

### 3. PortMarkers コンポーネント 🔵

**信頼性**: 🔵 *REQ-203*

ports配列からポート位置に小さな球/円柱を表示。medium別カラー（air=水色、water=青、refrigerant=緑）。direction別アイコン（in=矢印内、out=矢印外）。

**実装ファイル**: `packages/kuhl-viewer/src/components/renderers/parts/port-markers.tsx`

### 4. EquipmentRenderer メイン 🔵

**信頼性**: 🔵 *architecture.md*

useRegistry(node.id, node.type, ref)。useNodeEvents(node, node.type)。LOD判定（このタスクはLOD100のみ）。

**実装ファイル**: `packages/kuhl-viewer/src/components/renderers/equipment-renderer.tsx`
