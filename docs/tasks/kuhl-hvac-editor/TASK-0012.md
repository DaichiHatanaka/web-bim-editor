### 1. Viewer メインコンポーネント 🔵

**信頼性**: 🔵 *既存Viewerパターン*

R3F Canvas, WebGPU renderer。children injection パターン（Viewer Isolation）。Grid, OrbitControls, AmbientLight + DirectionalLight。

**実装ファイル**: `packages/kuhl-viewer/src/components/viewer.tsx`

### 2. レイヤー定数 🔵

**信頼性**: 🔵 *CLAUDE.md Three.jsレイヤー*

SCENE_LAYER=0, EDITOR_LAYER=1, ZONE_LAYER=2。レイヤー番号のハードコード禁止。

**実装ファイル**: `packages/kuhl-viewer/src/constants/layers.ts`

**信頼性**: 🟡 *既存パターンから推測*

node.type に基づいて適切なレンダラーコンポーネントを返す。未実装タイプにはフォールバック（BoxGeometry）。

**実装ファイル**: `packages/kuhl-viewer/src/components/renderers/node-renderer.tsx`
