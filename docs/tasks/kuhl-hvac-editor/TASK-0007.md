## タスク概要

HvacEquipmentBase を拡張した全13種の個別機器ノードスキーマを定義。AHU（詳細諸元: coolingCoil, heatingCoil, humidifier, heatRecovery等）、PAC（subType, refrigerant等）、Diffuser（subType 8種）は詳細実装。残り10種はベース拡張。

## 実装詳細

### 1. AhuNode（空調機）🔵

**信頼性**: 🔵 *interfaces.ts AhuNode*

id: `ahu_${string}`, type: 'ahu'。coolingCapacity, heatingCapacity, airflowRate, staticPressure, motorPower。coolingCoil(rows, fpi, enteringWaterTemp, leavingWaterTemp, waterFlowRate)。heatingCoil(type, capacity, waterFlowRate)。filterGrade, humidifier, heatRecovery, voltage, phase。

**実装ファイル**: `packages/kuhl-core/src/schema/nodes/ahu.ts`

### 2. PacNode・DiffuserNode 🔵

**信頼性**: 🔵 *interfaces.ts PacNode, DiffuserNode*

PacNode: `pac_${string}`, subType(5種), coolingCapacity, cop, refrigerantType, ratedPower, connectedIndoorUnitIds。
DiffuserNode: `diff_${string}`, subType(8種), airflowRate, effectiveArea, neckSize, throwDistance, noiseLevel, ceilingMounted。

**実装ファイル**: `packages/kuhl-core/src/schema/nodes/pac.ts`, `packages/kuhl-core/src/schema/nodes/diffuser.ts`

### 3. 残り10種の機器スキーマ 🔵

**信頼性**: 🔵 *interfaces.ts より*

FcuNode(`fcu_`), VrfOutdoorNode(`vrfo_`), VrfIndoorNode(`vrfi_`), DamperNode(`damp_`), FanNode(`fan_`), PumpNode(`pump_`), ChillerNode(`chlr_`), BoilerNode(`blr_`), CoolingTowerNode(`ct_`), ValveNode(`vlv_`)。各種 HvacEquipmentBase 拡張。
