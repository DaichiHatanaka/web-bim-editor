## 実装詳細

### 1. useViewer ストア 🔵

**信頼性**: 🔵 *architecture.md 3ストアアーキテクチャ*

SelectionPath: plantId, buildingId, levelId, zoneId, selectedIds[]。階層ガード: 親変更時に子リセット。カメラ状態、テーマ、表示トグル(grid, axes, labels)、LOD設定。

**実装ファイル**: `packages/kuhl-viewer/src/store/use-viewer.ts`

### 2. useEditor ストア 🔵

**信頼性**: 🔵 *設計文書 §4.1 フェーズ+ツールマッピング*

Phase: zone|equip|route|calc|takeoff。Mode: select|edit|delete|build。phaseTools マッピング。setPhase(phase): mode→select, tool→select にリセット。

**実装ファイル**: `apps/kuhl-editor/store/use-editor.ts`

## 単体テスト要件

### テストケース1: SelectionPath 更新・階層ガード 🔵

**Given**: buildingId を変更
**When**: setBuilding(newId)
**Then**: levelId, zoneId, selectedIds がリセット

### テストケース2: 表示トグル操作 🔵

**Given**: grid=true
**When**: toggleGrid()
**Then**: grid=false

### テストケース3: Phase 切替・Tool 自動リセット 🔵

**Given**: phase='zone', tool='zone_draw'
**When**: setPhase('equip')
**Then**: tool='select', mode='select'

### テストケース4: Mode 切替 🔵

**Given**: mode='select'
**When**: setMode('build')
**Then**: mode='build'

### テストケース5: phaseTools マッピング検証 🔵

**Given**: phase='zone'
**When**: getAvailableTools()
**Then**: ['select', 'zone_draw', 'zone_edit', 'load_calc']

**テストファイル**: `packages/kuhl-viewer/src/__tests__/store/use-viewer.test.ts`, `apps/kuhl-editor/__tests__/store/use-editor.test.ts`
