## 実装詳細

### 1. web-ifc 統合・IFCパース 🟡

**信頼性**: 🟡 *web-ifc APIの具体的な使い方は推測*

IfcAPI.Init() → OpenModel(buffer) → GetAllLines(modelId, ifcType) でジオメトリ抽出。IfcWall, IfcSlab, IfcBeam, IfcColumn, IfcCovering 対応。メインスレッド処理。

**実装ファイル**: `packages/kuhl-core/src/systems/ifc/ifc-import.ts`

### 2. ArchitectureRefRenderer 🔵

**信頼性**: 🔵 *REQ-006 + レンダラーパターン*

パース済みジオメトリをThree.jsメッシュで描画。raycastDisabled（編集不可）。半透明表示（建築躯体を参考表示）。

**実装ファイル**: `packages/kuhl-viewer/src/components/renderers/architecture-ref-renderer.tsx`

### 3. 階高・天井高取得（条件付き）🟡

**信頼性**: 🟡 *REQ-108 条件付き要件*

IfcBuildingStorey から Elevation を取得し、LevelNode に反映。取得できない場合はスキップ。
