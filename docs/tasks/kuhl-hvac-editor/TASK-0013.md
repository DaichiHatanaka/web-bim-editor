## 実装詳細

### 1. boundary → ShapeGeometry 変換 🔵

**信頼性**: 🔵 *REQ-101 + 既存Zone実装パターン*

boundary([[x,z]...]) から THREE.Shape を生成。ShapeGeometry で平面メッシュ作成。floorHeight の Y 座標に配置。

### 2. usage 別カラーマップ 🔵

**信頼性**: 🔵 *設計文書 §6 Phase 1*

office→青系、meeting→緑系、server_room→赤系、corridor→灰色等。MeshBasicMaterial(transparent: true, opacity: 0.3)。

### 3. ZONE_LAYER 配置・レジストリ登録 🔵

**信頼性**: 🔵 *CLAUDE.md レイヤー定義*

layers.set(ZONE_LAYER)。useRegistry(node.id, 'hvac_zone', ref)。useNodeEvents(node, 'hvac_zone')。

## 注意事項

- boundary は2D座標（[x,z]）だが、Three.js の ShapeGeometry は XY 平面なので回転が必要
- Renderer はジオメトリ生成を行わない原則だが、ゾーンは boundary からの直接生成が必要
